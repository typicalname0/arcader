## what u need
- php 7.1+
- composer (latest)
- mysqli
## how 2 install
you also need the silk icon pack found [here](http://www.famfamfam.com/lab/icons/silk/) and put it in /static/img/silk/
```
cd /your/web/directory/
git clone https://github.com/typicalname0/arcader.git
composer install
```
